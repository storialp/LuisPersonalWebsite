# Luis Personal Website

Personal site built with Astro, React, and Tailwind.

## Commands

Run commands from `C:\Users\luisp\Desktop\LuisPersonalWebsite`.

| Command | Action |
| :--- | :--- |
| `npm install` | Install dependencies |
| `npm run dev` | Start the local development server |
| `npm run build` | Create a production build in `dist/` |
| `npm run preview` | Preview the production build locally |
| `npm run lint` | Run Biome linting |
| `npm run format` | Format the codebase with Biome |
| `npm run check` | Run Biome checks with fixes |

## Build Cleanup Note

If you run a production build, delete the generated `dist/` folder afterward so build artifacts do not linger in the workspace.
